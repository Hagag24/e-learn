# درس: هيا نتعلم يا جدي — الصف الأول الابتدائي

مشروع ويب (HTML + CSS + JavaScript فقط) يعمل أوفلاين بدون مكتبات خارجية، مستوحى من واجهة Storyline، ويدعم اللمس والماوس.

## التشغيل
- افتح الملف `index.html` مباشرة في المتصفح، أو شغّل خادماً محلياً:
  - إن كان لديك Python: من داخل المجلد
    - Windows: `python -m http.server 5500`
    - ثم افتح: `http://localhost:5500/`

## بنية الملفات
- `index.html`: يحتوي على كل الشاشات (المقدمة + الأنشطة 1–5) والتنقل.
- `styles.css`: تصميم، ألوان، أنيميشن، تجاوب.
- `app.js`: منطق التفاعل، تشغيل الأصوات، التغذية الراجعة، الكونفيتي.
- `assets/images/`: ضع الصور هنا بصيغة PNG بالأسماء العربية:
  - `باب.png`, `تفاح.png`, `ثعلب.png`, `بيت.png`, `أسد.png`, `ديك.png`, `كلب.png`
- `assets/sounds/`: ضع الأصوات هنا بصيغة MP3:
  - `correct.mp3`, `wrong.mp3`, `cheer.mp3`, `ba.mp3`, `ta.mp3`, `tha.mp3`

ملاحظة: إن لم تتوفر الصور/الأصوات، يعتمد التطبيق فولبك داخلي:
- صور SVG نصية بالرموز التعبيرية والعنوان العربي.
- صوت توليدي بسيط عبر WebAudio.

## الأنشطة
1. **اكتشف الحرف المختلف**: بطاقات (بـ – تـ – ثـ). الصحيح هو (ثـ).
   - الصحيح: يضيء بالأخضر + نص "أحسنت!" + نجوم + صوت صحيح.
   - الخطأ: يهتز + "جرب مرة أخرى" + صوت خطأ.
2. **استمع وانطق**: دوائر للحروف (باء – تاء – ثاء) مع زر تشغيل الصوت.
   - بعد تشغيل الثلاثة: رسالة نجاح + نجوم + تصفيق.
3. **اختر الحرف الأول للكلمة**: صور (تفاح، باب، ثعلب) وخيارات (ب، ت، ث).
   - الصحيح: نجمة + صوت "أحسنت".
   - الخطأ: اهتزاز + "راجع الفيديو".
4. **اقطع الكلمة إلى مقاطع**: كلمة "بيت" → (بيـ | تـ) عند الضغط على المقص، وزر لإعادة.
   - تغذية راجعة لكل حالة + مؤثرات Zoom ووميض.
5. **صوت البداية**: صور (أسد، ديك، كلب) وخيارات الأصوات الأولى.
   - الصحيح: يتحول الزر أخضر + "صحيح".
   - الخطأ: اهتزاز + "خطأ".

## الخطوط
- تم استخدام `Cairo`, `Noto Kufi Arabic` إن كانت مثبتة. لإضافة خطوط محلية:
  - ضع ملفات الخط داخل مجلد محلي وأضف `@font-face` في `styles.css`.

## التخصيص
- استخدم متغيرات CSS أعلى `styles.css` لتعديل الألوان بسرعة.
- يمكن تعديل النصوص بسهولة داخل `index.html`.

## قيود وملحوظات
- لا توجد مكتبات خارجية.
- المشروع يعمل أوفلاين بالكامل.
- الواجهات متجاوبة والأزرار كبيرة مناسبة للأطفال.

## الترخيص
- لأغراض تعليمية داخل الصف.
